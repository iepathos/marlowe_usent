A simple feature selection method that filters features according to a feature score. It uses a feature score (instance of FeatureScore) to eliminate features in one of three possible modes:

  • keep a specified number of features [default]
  • eliminate all features whose score is below some threshold
  • eliminate all features whose score is a certain number of standard deviations above that obtained using random labels
